문제 설명

두 개의 행렬을 입력받아, 곱셈 결과값을 출력하기


문제 분석

다른 요소보다, 행렬의 곱 자체의 흐름(하나의 행렬 원소에 대해, 첫 번째 행렬에서는 가로로, 두 번째 행렬에서는 세로로 움직이며 곱셈하고 더하는 코드임을 기억하자.)

*코드-Python*

```python
def solution(arr1, arr2):
    
    r = len(arr1)
    c1 = len(arr1[0])
    c2 = len(arr2[0])
    
    answer = []
    
    for i in range(r):
        temp = []
        for j in range(c2):
            temp.append(0)
        answer.append(temp)
    
    
    
    for i in range(r):
        for j in range(c2):
            for k in range(c1):
                answer[i][j]+=arr1[i][k]*arr2[k][j]
    

    return answer
 
```
