
# [99클럽 코테 스터디] 19일차 TIL - 거리두기 확인하기


문제 설명

맨해튼 거리 2이하에 위치하고, 참가자 사이에 파티션이 존재하지 않는 경우를 거리두기가 지켜지지 않았다고 할 때, 각 케이스에 대해 거리두기가 지켜지고 있는지 판단하기

문제 분석

각 케이스 별로 if문을 사용해서 해결하는 방법을 고민해 보았는데, 오히려 엣지 케이스를 꼼꼼하게 잡아내지 못할 것 같다. 대신, 거리두기의 성립 조건이 미로찾기에서 거리가 2 이하로 도달할 수 있는지를 체크하는 것과 유사하게 모델링하여 푸는 방법을 사용했다.

*코드-Python*

```Python
from collections import deque
def solution(places):
    answer = []
    
    
    
    for place in places:
        
        starts = []
        
        for i in range(5):
            for j in range(5):
                if place[i][j] == "P":
                    
                    starts.append([i,j])
        
        flag = True
                    
        for start in starts:  
            distances = []
            for i in range(5):
                    temp = []
                    for j in range(5):
                        temp.append(float("inf"))
                    distances.append(temp)
            
            visited = [] 

            for i in range(5):
                temp = []
                for j in range(5):
                    temp.append(False)
                visited.append(temp)



            q = deque()
            startr = start[0]
            startc = start[1]
            q.append([startr,startc])
            distances[startr][startc] = 0

            while q:
                cur = q.popleft()
                r = cur[0]
                c = cur[1]
                visited[r][c] = True

                if r-1 >= 0:
                    if visited[r-1][c] == False:
                        if place[r-1][c] != "X":
                            q.append([r-1, c])
                            distances[r-1][c] = min(distances[r-1][c], distances[r][c]+1)
                            if place[r-1][c] == "P" and distances[r-1][c] <= 2:
                                break
                            
                

                if r+1 < 5:
                    if visited[r+1][c] == False:
                        if place[r+1][c] != "X":
                            q.append([r+1, c])
                            distances[r+1][c] = min(distances[r+1][c], distances[r][c]+1)
                            if place[r+1][c] == "P" and distances[r+1][c] <= 2:
                                break

                if c-1 >= 0:
                    if visited[r][c-1] == False:
                        if place[r][c-1] != "X":
                            q.append([r, c-1])
                            distances[r][c-1] = min(distances[r][c-1], distances[r][c]+1)
                            if place[r][c-1] == "P" and distances[r][c-1] <= 2:
                                break

                if c+1 < 5:
                    if visited[r][c+1] == False:
                        if place[r][c+1] != "X":
                            q.append([r, c+1])
                            distances[r][c+1] = min(distances[r][c+1], distances[r][c]+1)
                            if place[r][c+1] == "P" and distances[r][c+1] <= 2:
                                #print(place, start, r, c+1)
                                break
            if q:
                flag = False
        if flag == True:
            answer.append(1)
        else:
            answer.append(0)
        
    
    return answer
```
